## 📖 Overview
This is a beginner-friendly **Python project** that simulates a train ticket booking system using **Object-Oriented Programming (OOP)** concepts.  
It allows users to:
- Enter passenger details  
- Book tickets with random train numbers  
- Check train status  
- Generate ticket fare dynamically  
- Print a formatted train ticket  

---

## ✨ Features
- 🎫 **Ticket Booking** – Generates a random train number and confirms booking.  
- ⏱️ **Train Status** – Displays whether the train is on time.  
- 💰 **Fare Calculation** – Random fare between Rs 500–2000.  
- 🧾 **Ticket Printing** – Shows passenger name, train number, route, and fare.  
- 📍 **Interactive Input** – User enters source, destination, and passenger name.  

---

## 🛠️ Technologies Used
- **Python 3**  
- **Random module** for dynamic train numbers and fares  
- **Datetime module** (optional improvement) for booking date/time
