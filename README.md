## 📌 Project Overview

The **Train Ticket Booking System** is a Python console application that simulates a simple railway ticket booking process.

This project demonstrates the use of **Object-Oriented Programming (OOP)** along with Python modules like `random` and `datetime` to generate realistic booking information.

It is designed as a beginner-to-intermediate project for learning Python classes, objects, methods, and modular programming.

---

# ✨ Features

✅ Book Train Tickets

✅ Automatic Train Number Generation

✅ Random Seat Allocation

✅ Random Ticket Fare Generation

✅ Live Booking Date & Time

✅ Passenger Information

✅ Train Status

✅ Beautiful Ticket Display
